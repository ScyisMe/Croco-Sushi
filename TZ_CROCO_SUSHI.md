# Технічне завдання (ТЗ) на розробку сайту Croco Sushi

## 1. ЗАГАЛЬНА ІНФОРМАЦІЯ

### 1.1. Мета проекту
Розробка повнофункціонального сайту доставки суші "Croco Sushi" з функціоналом аналогічним до king-roll.com.ua, включаючи:
- Каталог продуктів з категоріями
- Кошик та оформлення замовлень
- Систему доставки
- Адмін-панель для управління
- Інтеграції для платежів та доставки
- Максимальну безпеку та стабільність
- Високу продуктивність та масштабованість

### 1.2. Назва проекту
**Croco Sushi** - сервіс доставки суші

### 1.3. Технологічний стек (на основі наявного проекту)
- **Backend:** FastAPI + PostgreSQL + SQLAlchemy (async)
- **Frontend:** React/Next.js 14+ (App Router)
- **База даних:** PostgreSQL 17
- **Кешування:** Redis 7
- **Завдання:** Celery
- **Міграції:** Alembic
- **Контейнеризація:** Docker & Docker Compose
- **Моніторинг:** Sentry, Prometheus + Grafana
- **CI/CD:** GitHub Actions / GitLab CI

### 1.4. Дизайн та UI/UX (на основі king-roll.com.ua)

Дизайн сайту Croco Sushi повинен бути ідентичним або максимально схожим на дизайн сайту [king-roll.com.ua](https://king-roll.com.ua/), включаючи:
- Кольорову схему та стилістику
- Розташування елементів
- Типографіку
- Анімації та переходи
- Адаптивність для мобільних пристроїв

## 2. ДИЗАЙН ТА UI/UX ВИМОГИ

### 2.0. Загальні вимоги до дизайну

#### 2.0.1. Кольорова палітра (на основі king-roll.com.ua)
- **Основний колір:** Зелений (#00A859 або схожий) - для акцентів, кнопок, логотипу
- **Фон:** Білий (#FFFFFF) - для основного фону
- **Текст:** Темно-сірий (#333333) - для основного тексту
- **Вторинний текст:** Сірий (#666666) - для описів та підписів
- **Акцентний колір:** Червоний (#FF0000 або схожий) - для акцій та хітів
- **Бордер:** Світло-сірий (#E5E5E5) - для розділювачів та рамок
- **Hover ефекти:** Темніший відтінок основного кольору

#### 2.0.2. Типографіка
- **Заголовки:** Жирний шрифт, розмір від 24px до 48px
- **Підзаголовки:** Напівжирний шрифт, розмір 18px-24px
- **Основний текст:** Звичайний шрифт, розмір 14px-16px
- **Дрібний текст:** Розмір 12px-14px для підписів та описів
- **Шрифт:** Sans-serif (наприклад, Inter, Roboto, або системний)

#### 2.0.3. Адаптивність
- **Desktop:** Від 1200px і більше
- **Tablet:** 768px - 1199px
- **Mobile:** До 767px
- **Мобільна версія:** Горизонтальне меню перетворюється на hamburger меню
- **Кошик:** На мобільних пристроях відкривається як модальне вікно на весь екран

#### 2.0.4. Анімації та переходи
- **Тривалість переходів:** 200-300ms для стандартних елементів
- **Hover ефекти:** Плавна зміна кольору та тіні
- **Завантаження:** Skeleton loaders для контенту
- **Модальні вікна:** Плавна поява з затемненням фону
- **Кнопки:** Ефект натискання (active state)
